About This Project

This pet detection app uses a YOLOv8 object detection model trained on pet images to identify cats and dogs in uploaded photos.

How to Use

  • Go to the Home page.
  • Upload a clear image of a pet (cat or dog).
  • Wait for the model to process and detect.
  • View the image with bounding boxes and labels.

Model Info

The model was trained using YOLOv8 with a custom dataset on Google Colab. It identifies pets with high accuracy.
